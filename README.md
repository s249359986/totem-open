## 服务端

### 使用



``` 
安装 pm2 
npm install pm2 -g

安装依赖
npm install

启动项目
npm run start

```

### 目录

```
- service 根目录
 - app.js 主文件
 - routes 路由 
 - serve 服务
 - sources 临时资源文件
 - test 单测
 - tool 工具

```

## 客服端

### 使用

```
安装依赖
npm install

启动项目
npm run serve

打包项目
npm run build

```
### 目录

```
- web 根目录
- src 源文件
 - views
  - home (图片压缩)
  - qr (二维码生成)


```
